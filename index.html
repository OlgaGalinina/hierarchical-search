How to Configure Hierarchical Search: Closed-form Scaling Laws for 6G Beam Management

Olga Galinina  ·  Sergey Andreev  ·  Robert Heath

Hierarchical search is the standard alternative to exhaustive search in directional wireless access. Yet its performance depends on how it is configured. This page illustrates the choice of optimal parameters and provides a simple baseline for 6G beam management research.

Note. In reality, misdetection probability has two regimes:
  • SNR-limited, where the target beam has high gain and non-target beams have near-zero gain. This widget assumes the SNR-limited regime.
  • Sidelobe-limited, where adjacent beam sidelobes are large and even without noise, the faded signal on a sidelobe can exceed the main lobe. The above model ignores this regime, assuming beams have no overlapping sidelobes.
Key results

Closed-form optimal configuration

Propositions 1 - 3
Unconstrained optimum
Basic configuration (no inter-layer intervals \(c\)):
\( L^* = \ln X \) layers
\( N^* = X^{1/\ln X} \) beams per layer
\( R^* = e\ln X \) total resources
Option A delay (equal inter-layer intervals \(c\)):
\( L_A^* = \dfrac{\ln X}{1 + W(c/e)} \) layers
Option B delay (fixed period \(c_1\) between sweep starts):
\( L_B^* = \dfrac{\ln X}{2\,W\!\left(\dfrac{\sqrt{c_1 \ln X}}{2}\right)} \) layers
Extended proposition
With reliability constraint (Rician channel model)
Per-layer beam misdetection probability (exact, Rician fading):
\( P_e(N,\theta) = 1 - \sum \limits_{m=0}^{N-1}\binom{N-1}{m}(-1)^m \dfrac{1}{1+m\sigma^2_{\mathrm{tot}}/N_0} \exp\!\left(-\dfrac{m\nu/N_0}{1+m\sigma^2_{\mathrm{tot}}/N_0}\right), \)
where ν=GPrxKK+1 is the LOS power and σtot2=GPrxK+1+N0 is the total variance, including scattering and receiver noise. This expression is exact for all SNR and all K≥0.

At typical 6G SNR values, the reliability constraint may be inactive. The calculator shows when it activates.

Assumptions: boundary-centered beam placement (HPBW = \(\theta/(N{-}1)\)), Rician \(K\)-factor with i.i.d. scatter, AWGN noise with figure \(N_F\), layer independence. The gain on the correct beam is \(G = \min(N_\mathrm{ant},\, 101.5^\circ(N{-}1)/\theta)\), capped at the array size; \(\nu\) and \(\sigma^2_{\mathrm{tot}}\) are computed from this gain and the Rician/SNR parameters.

# Exact per-layer error probability — Proposition 1
# Recursive binomial coefficient update avoids math.comb overflow.
# Numerically stable for N ≤ 48; use Pe_exact_quad for larger N.

import math
from scipy.special import i0
from scipy.integrate import quad

def beam_gain(N, theta_deg, Nant=128):
    return min(Nant, 101.5 * (N - 1) / theta_deg)

def Pe_exact(N, theta_deg, SNR, K=10, Nant=128):
    """Alternating binomial sum with recursive coefficient update."""
    G  = beam_gain(N, theta_deg, Nant)
    nu = G * SNR * K / (K + 1)
    s2 = G * SNR / (K + 1) + 1.0
    s, binom, sign = 1.0, 1.0, -1.0
    for m in range(1, N):
        binom *= (N - m) / m
        denom  = 1.0 + m * s2
        lt     = math.exp(-m * nu / denom) / denom
        s     += sign * binom * lt
        sign  *= -1.0
    pe = 1.0 - s
    return pe if 0.0 <= pe <= 1.0 else float('nan')

def Pe_exact_quad(N, theta_deg, SNR, K=10, Nant=128):
    """Numerically stable for all N. Uses scipy quad — no cancellation."""
    G  = beam_gain(N, theta_deg, Nant)
    nu = G * SNR * K / (K + 1)
    s2 = G * SNR / (K + 1) + 1.0
    def integrand(y):
        if y <= 0: return 0.0
        arg    = 2.0 * math.sqrt(nu * y) / s2
        log_i0 = (arg - 0.5*math.log(2*math.pi*arg)
                  if arg > 500 else
                  math.log(float(i0(arg))) if arg > 0 else 0.0)
        return ((1 - math.exp(-y))**(N-1) *
                math.exp(-math.log(s2) - (nu+y)/s2 + log_i0))
    upper  = max(50.0, 30*s2 + 30*nu)
    hints  = [s2, 2*s2] + ([nu] if nu > 0.1 else [])
    val, _ = quad(integrand, 0, upper, limit=200,
                  epsabs=1e-12, epsrel=1e-10, points=hints)
    return 1.0 - val
# Harmonic approximation — valid when G(N,θ)·SNR ≫ K+1
# Underestimates Pe; useful for analytical insight into ln N scaling.

def harmonic(n):
    return sum(1.0/k for k in range(1, n+1)) if n >= 1 else 0.0

def Pe_harmonic(N, theta_deg, SNR, K=10, Nant=128):
    """Pe ≈ (K+1)·exp(-K)·H_{N-1} / (G·SNR)"""
    G = beam_gain(N, theta_deg, Nant)
    return (K+1) * math.exp(-K) * harmonic(N-1) / (G * SNR)

# For N in [2, 128]:
# Pe_harmonic underestimates Pe_exact by 10–80% depending on G·SNR.
# The approximation improves as G·SNR / (K+1) grows.
# Optimal number of layers — Propositions 1, 2, 3
# Uses scipy.special.lambertw for the Lambert-W function.

import math
from scipy.special import lambertw

def L_opt_resource(X):
    """Proposition 1: resource-optimal depth (c=0)."""
    return math.log(X)

def L_opt_A(X, c):
    """Proposition 2 (Option A): equal inter-layer intervals c."""
    W = lambertw(c / math.e).real
    return math.log(X) / (1 + W)

def L_opt_B(X, c1):
    """Proposition 3 (Option B): fixed period c1 between sweep starts."""
    if X <= 1: return 1.0
    arg = math.sqrt(c1 * math.log(X)) / 2.0
    W   = lambertw(arg).real
    return math.log(X) / (2.0 * W) if W > 0 else 1.0

def N_opt(X, L):
    """Optimal beams per layer for given L."""
    return X ** (1.0 / L)

# Example: X=128, c1=74 (NR numerology 0, 10 ms burst)
# L* ≈ L_opt_B(128, 74) ≈ 2.4 → use L=2 or L=3
# N* ≈ N_opt(128, 2.4) ≈ 6.7 → use N=7
# Reliability constraint check — exact and harmonic
# Option B: sector narrows as theta_ell = theta0/(N-1)^{ell-1}

def geo_sum(N, L):
    """sum_{ell=1}^{L} 1/(N-1)^ell"""
    if N - 1 <= 1: return float(L)
    return (1.0 - 1.0/(N-1)**L) / (N - 2)

def reliability_exact(L, X, SNR, K=10, theta0=120.0,
                         Nant=128, epsilon=0.01):
    """prod_{ell=1}^{L} (1-Pe_exact(ell)) "">"" 1-epsilon"""
    N = X**(1.0/L)
    if N < 2: return False
    log_prod = 0.0
    for ell in range(1, L+1):
        th  = theta0 / (N-1)**(ell-1)
        pe  = Pe_exact_quad(N, th, SNR, K, Nant)
        log_prod += math.log(max(1.0 - pe, 1e-15))
    return log_prod >= math.log(1.0 - epsilon)

def reliability_harmonic(L, X, SNR, K=10, theta0=120.0,
                           Nant=128, epsilon=0.01):
    """Closed-form sufficient condition (harmonic approx, high SNR)."""
    N   = X**(1.0/L)
    if N < 2: return False
    H   = harmonic(max(1, round(N)-1))
    lhs = (K+1)*math.exp(-K)*H*theta0/(101.5*SNR)*geo_sum(N, L)
    return lhs <= epsilon
